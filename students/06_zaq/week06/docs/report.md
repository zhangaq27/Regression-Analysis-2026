# Week06 实验报告：回归引擎与多市场分析

## 一、实验概述

本次实验实现了一个手写的 OLS 回归引擎 `CustomOLS`，采用**面向对象（Class）**封装，支持拟合、预测、R² 计算和一般线性假设 F 检验。该引擎被应用于：
1. 合成数据白盒测试（验证正确性）
2. 真实市场营销数据（北美 vs 欧洲市场多实例分析）

## 二、Task 1：推断引擎实现

### 选择：面向对象（Class）方式

**核心方法实现：**
- `fit()`: 计算 β_hat = (X^T X)^{-1} X^T y，同时计算 σ² 和协方差矩阵
- `predict()`: 返回 X @ coef_
- `score()`: 返回 R² = 1 - SSE/SST
- `f_test()`: 执行一般线性假设检验 Cβ = d

**截距项处理：**
在数据预处理阶段通过 `add_intercept()` 函数在 X 矩阵左侧添加一列全 1，模型内部不做特殊处理。这样做的好处是：
- 模型实现更简洁统一
- 截距项与其他系数同等对待
- 便于进行包含截距的假设检验

### 为什么选择 Class？

在 Task 3 中需要同时处理两个独立市场（北美和欧洲）。使用 Class 可以：
- 每个市场创建独立的模型实例（`model_na` 和 `model_eu`），互不干扰
- 系数、协方差矩阵等状态被完美封装在实例内部
- 代码更简洁，易于扩展和维护
- 符合真实业务场景（多个市场独立建模）

## 三、Task 2：通用评价函数

实现了 `evaluate_model()` 函数，利用 Python 的**鸭子类型（Duck Typing）**特性：
- 不检查模型的具体类型
- 只要求模型有 `.fit()`, `.predict()`, `.score()` 方法
- 无论是 `CustomOLS` 还是 `sklearn.LinearRegression` 都能自动工作

## 四、Task 3：双重数据训练

### 场景 A：合成数据白盒测试
- 生成 500 条数据，真实系数 β = [3, 2, -1]（截距 + 2个特征）
- CustomOLS 与 sklearn 结果一致，R² ≈ 0.85-0.90
- 断言验证通过

### 场景 B：真实数据（北美 vs 欧洲）
- 读取 `q3_marketing.csv` 数据
- 分别建立 `model_na` 和 `model_eu` 两个独立实例
- 执行 F 检验验证广告投放策略的有效性

#### F 检验结果解读
- **原假设 H₀**: 所有特征系数同时为 0（广告投放无效）
- **备择假设 H₁**: 至少有一个特征系数不为 0（广告有效）
- 若 p < 0.05，拒绝原假设，认为广告投放策略显著有效
